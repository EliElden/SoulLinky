# Soul Linky
* Разработчики: Литвиненко М.И. и Морозова Э.Б.
* Проект: Telegram-бот для парочек


## Концепция и решаемая проблема
* Суть: Создание закрытого и удобного пространства для двоих в Telegram.
* Механика: Связывание двух пользователей в единую систему с помощью уникальных ID.
* Цель: Упростить коммуникацию, синхронизировать планы и сохранить важные моменты пары в одном месте, а также разнообразить отношения на расстоянии


## Ключевой функционал
* Mood Tracker: Ежедневное отслеживание и обмен настроением с партнером.
* Общий вишлист: Синхронизированный список желаний и идей для подарков.
* Система уведомлений: Автоматические напоминания о годовщинах и важных совместных датах.
