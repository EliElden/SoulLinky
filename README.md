**SoulLinky** — это специализированный Telegram-бот, разработанный для пар. Проект решает проблему разрозненности коммуникации, помогая партнерам синхронизировать планы, делиться эмоциями и сохранять важные совместные моменты в едином, закрытом и безопасном пространстве. Особенно актуален для поддержки и разнообразия отношений на расстоянии.

---

## 🎯 Концепция и решаемая проблема

Современные пары используют множество приложений для общения, планирования и трекинга привычек. **Цель SoulLinky** — объединить эти механики в привычном и доступном интерфейсе Telegram. 

**Суть механики:** Связывание двух независимых пользователей в единую закрытую систему (через уникальные числовые ID или username) с системой взаимного подтверждения (handshake). После образования пары пользователям открывается эксклюзивный функционал для взаимодействия друг с другом.

---

## ✨ Ключевой функционал (Текущий релиз)

- 🔗 **Парное подключение:** Надежная связь двух пользователей через ID или `@username` со строгой системой взаимного подтверждения запроса.
- 💌 **Любовные послания:** Интерактивный обмен контентом (поддержка текстовых сообщений, фото, голосовых заметок и видеосообщений-«кружочков»).
- 🐱 **Мемы с котиками:** Уникальная функция автоматического наложения текста любовного послания на случайное изображение котенка (динамическая генерация «на лету» с помощью библиотеки `Pillow`).
- 🎁 **Общий вишлист:** Синхронизированный инструмент для совместного создания и управления списками желаний, подарков и идей для будущих свиданий.
- 📅 **Важные даты:** Логирование годовщин и совместных событий (однократных или ежегодных) с автоматическими напоминаниями, работающими в фоновом потоке.
- 🎭 **Муд-трекер:** Ежедневное отслеживание психоэмоционального состояния партнеров, агрегация статистики эмоций и ведение детального журнала изменений.
- 🔥 **Серия (Streak):** Игровая механика (геймификация), подсчитывающая количество дней непрерывного общения через систему посланий.
- 🛑 **Таймаут («В угол»):** Шуточная модерация внутри пары — возможность временной блокировки отправки посланий партнером (на 5, 15 или 60 минут) за «плохое поведение».
- 🛡 **Безопасность и приватность:** Наличие личного черного списка для блокировки нежелательных пользователей и функция мгновенного безопасного разрыва связи с текущим партнером.
- 🛠 **Панель администратора:** Выделенный инструментарий для разработчиков, включающий массовую рассылку системных новостей и динамическую статистику использования бота.

## 🛠 Стек технологий

- **Язык программирования:** Python 3.10+
- **Фреймворк:** `pyTelegramBotAPI` (Telebot) — асинхронное взаимодействие с Telegram Bot API.
- **Обработка графики:** `Pillow` (PIL) — динамический рендеринг текста на изображениях.
- **База данных:** `SQLite` (модуль `db.py`) — легковесное реляционное хранилище для связей, логов настроения, вишлистов и дат.
- **Многопоточность:** Встроенный модуль `threading` — обеспечение автономной работы фонового планировщика напоминаний (`reminder_loop`).
- **Архитектура:** Объектно-ориентированный подход (ООП). Состояния пользователей инкапсулированы в `BotStateManager`, а генерация интерфейсов вынесена в структуру `BotInterfaceManager`.
